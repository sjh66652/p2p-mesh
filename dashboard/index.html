🚀 一键部署

🖥️ 服务端部署
在 VPS/服务器上一键部署
Linux
# 克隆项目 git clone <repo> /opt/p2p-mesh cd /opt/p2p-mesh # 一键部署 (需要 root) sudo bash deploy.sh server --domain mesh.yourdomain.com # 或使用交互式菜单 sudo bash deploy.sh
Step 1 安装 Docker + 依赖
Step 2 生成安全密钥
Step 3 构建 Docker 镜像
Step 4 启动所有服务
Step 5 配置 SSL + 防火墙
Step 6 健康验证
💻 客户端部署
在终端设备上一键部署
Linux Windows
🐧 Linux
# 安装并配置 bash scripts/deploy-client.sh \ --server https://mesh.yourdomain.com \ --token <your_auth_token> # 查看状态 systemctl status p2p-mesh-tunnel
🪟 Windows (PowerShell 管理员)
# 部署客户端 .\scripts\deploy-client.ps1 ` -Server "https://mesh.yourdomain.com" ` -Token "<token>" # 查看状态 Get-Service P2PMeshTunnel
🐳 Docker (任意平台)
# 构建客户端镜像 docker build -f deployment/Dockerfile.tunnel \ -t p2p-mesh-tunnel data-plane/ # 运行 docker run -d --network host \ -e API_URL=https://mesh.yourdomain.com \ -e AUTH_TOKEN=<token> \ p2p-mesh-tunnel
⚡ 快速体验 (开发环境)
本地一键启动所有服务，无需配置
# 方式 1: 使用总控脚本 bash deploy.sh quick # 方式 2: 直接使用 docker compose cd deployment docker compose -f docker-compose.microservices.yml --env-file .env up -d
⚠️ 此模式仅用于开发和体验，自动生成弱密钥，不应用于生产环境。
